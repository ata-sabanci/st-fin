The Strategy System

Twenty condition slots, a quorum-based signal engine, and unified position management. Every parameter accessible through the settings panel — no code editing required.

⚡ Condition Slots

The strategy provides 20 independent condition slots. Each slot monitors any one of the 20 geometric lenses and tests whether its current value meets a specified condition. Each slot has five configurable settings:

Variable

Which of the 20 lenses this slot watches. OFF = slot excluded.

Condition

How the value is evaluated: >, <, between, cross >, cross <

Threshold

Numeric level for the test. Continuous — any value, not a preset list.

Apply to

Long entry, Short entry, Both entry, Long exit, Short exit, Both exit, or All.

700 configurations per slot. 20 variables × 5 conditions × 7 sides = 700 structural configurations from a single active slot. This number is specific to this framework — it comes from geometry, not a textbook formula.

⚙️ Quorum Logic

A single setting — Minimum slots required — controls how many active slots must pass their condition before a signal fires. This creates a flexible spectrum:

Min = 1

OR logic — any single slot is enough. Most permissive, most signals.

Min = 3 of 5

Quorum — majority must agree. Balanced filtering.

Min = All

AND logic — every slot must pass. Most restrictive, fewest signals.

Conflict guard: If both Long and Short entry quorums are met on the same bar, neither trade is executed. Market indecision, not a directional signal.

Signal Modes

Close mode: Calculations use the previous bar's confirmed data. Signals appear only after a bar completes — what you see is final. Recommended for backtesting.

Live mode: Calculations update in real time as the current bar forms. Faster, but signals may shift before the bar closes.

🛡️ Position Management

Three independent tools for managing open positions. Each can be enabled separately, and all three can run simultaneously.

Fixed Stop-Loss

Percentage below entry (long) or above entry (short). Calculated from the actual average entry price, not from the signal bar's close.

Fixed Take-Profit

Percentage above entry (long) or below entry (short). Same calculation from actual fill price. Whichever level is hit first closes the position.

Trailing Stop

Tracks the best price since entry. Long: follows highest high, stops at trail% below peak. Short: follows lowest low, stops at trail% above trough. Only moves in your favor.

Unified exit: When both fixed SL and trailing stop are active, the strategy automatically selects the most protective level. A single exit order is maintained at all times.
